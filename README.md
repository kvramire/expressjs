# expressjs
clone repository
==========================
> git clone https://github.com/kvramire/expressjs.git
> git add .
> git commit -m "comment"
> git push

Install express
=============================
> npm install express --save
> npm install nodemon  
> node index.js
> nodemon index.js

Express - basics
==================
> separate routes implementation into multiple controllers
> passing params through url

Express configurations
==========================
